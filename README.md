# Körjournal 🚗

En enkel och stilren Progressive Web App (PWA) för att logga dina bilresor i realtid.

## Funktioner

- **Realtidsspårning**: Mäter tid och sträcka (km) under resans gång.
- **GPS-integration**: Använder enhetens GPS för exakt avståndsmätning.
- **Trip Historik**: Spara dina tidigare resor lokalt på enheten.
- **Screen Wake Lock**: Håller skärmen vaken under pågående resa (förutsatt stöd i webbläsaren).
- **Dimmer-läge**: Sänker ljusstyrkan på gränssnittet under resan för att spara batteri och minska distraktion.
- **Mörkt tema**: Optimerat för användning i bilen.

## Teknologier

- **HTML5 & CSS3** (Vanilla)
- **Javascript** (Vanilla)
- **Geolocation API**
- **Screen Wake Lock API**
- **LocalStorage** for datalagring

## Kom igång

1. Ladda ner eller klona detta repository.
2. Öppna `index.html` i en modern webbläsare.
3. Klicka på "STARTA RESA" för att börja logga.

## Installation som App (PWA)

Eftersom detta är en PWA kan du installera den direkt på din startskärm (t.ex. på iPhone via "Lägg till på hemskärmen" i Safari).
